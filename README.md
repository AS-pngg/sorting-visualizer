# Sorting Visualizer

Interactive sorting algorithm visualizer built with React, Tailwind CSS and Framer Motion.

## Features

- Bubble Sort
- Selection Sort
- Insertion Sort
- Merge Sort
- Quick Sort
- Heap Sort
- Animated visualizations
- Adjustable speed
- Adjustable array size

## Tech Stack

React  
Tailwind CSS  
Framer Motion
